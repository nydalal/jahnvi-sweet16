STEP 1: GitHub Account
  □ Go to github.com
  □ Sign up (free)
  □ Verify email

STEP 2: Netlify Account
  □ Go to netlify.com
  □ Sign up with GitHub
  □ Authorize GitHub connection

STEP 3: Create GitHub Repository
  □ On GitHub: + → New repository
  □ Name: jahnvi-sweet16
  □ Description: Jahnvi's Sweet Sixteen Celebration Website
  □ Create repository
  □ Upload ALL files from /home/claude/ to this repo

STEP 4: Connect to Netlify
  □ On Netlify: Add new site → Import an existing project
  □ Select GitHub
  □ Choose jahnvi-sweet16 repo
  □ Click Deploy
  □ WAIT 2-3 MINUTES for build

STEP 5: You're LIVE! 🎉
  □ Netlify gives you a URL
  □ Share it with guests
  □ Website is FREE & FOREVER
